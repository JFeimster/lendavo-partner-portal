In 2026, algorithmic underwriting is rejecting **54% of small business applications**. Traditional financial referrals leak "Trust Equity" and burn relationships. 

This portal acts as an **Active Utility**—an "Engineering-as-Marketing" tool. It leverages an interactive UI and psychological triggers (Loss Aversion, High-Status Framing, and System Momentum) to compel CPAs, Coaches, and Loan Officers to utilize the Lendavo AI Diagnostic tool.

## ⚙️ Core Architecture & Features

- **The "Bento Grid" Layout:** Complex financial mechanisms (like EIN/SSN decoupling) are contained in high-contrast, easy-to-digest glass-morphic containers to reduce cognitive friction.
- **Trust Equity Leakage Calculator:** A dynamic, interactive JS module that gamifies loss aversion by showing advisors exactly how much capital—and trust—they are losing to the 54% rejection wall.
- **System Status "Social Proof":** Subtle, real-time terminal-style notifications that create system momentum without looking like cheap, intrusive pop-ups.
- **Luxury-Futurism Design System:** - `bg-brand-dark`: `#050505` (Deep Space Dark Mode)

  - `text-brand-neon`: `#00F0FF` (Terminal Cyan)
  - Typography: *Space Grotesk* (Display/Headers) & *Inter* (Body Content)
